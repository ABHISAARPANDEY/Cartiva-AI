  - E-commerce: Blue/Cyan tones
  - Finance: Violet/Purple tones

  - Real Estate: Emerald/Teal tones

### Style Requirements
- **Modern & Clean:** Minimal, professional design
- **Consistent:** All images should follow same design language
- **High Quality:** 4K resolution where possible, optimized for web
- **Accessible:** Good contrast, readable text overlays
- **Brand-Aligned:** Match website's dark theme and modern aesthetic

### Technical Specifications
- **Format:** PNG (for graphics with transparency) or WebP (for optimized web)
- **Resolution:** Minimum 1920px width for hero images, 1200px for infographics
- **File Size:** Optimized for web (under 500KB per image when possible)
- **Aspect Ratios:**

  - Infographics: 4:3 or 16:9
