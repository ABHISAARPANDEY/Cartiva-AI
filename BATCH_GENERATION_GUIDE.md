## 💡 PRO TIPS FOR BATCH GENERATION

### 1. **Use Consistent Style Keywords**
Add these to EVERY prompt for consistency:
- "modern professional design"
- "high quality 4K"
- "clean design"
- Industry-specific color tones

### 2. **Generate in Order**
- Start with one industry batch (8 images)
- Review quality before moving to next batch
- Adjust prompts if needed based on results

### 3. **Save with Clear Names**
Use the exact filenames specified so they match your code imports

### 4. **Generate Variations**
For critical images (hero, dashboard), generate 2-3 variations and pick the best

### 5. **Test After Each Batch**
- Generate one batch
- Test on website
- Adjust style if needed
- Continue with next batch
